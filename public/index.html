Built on proven cryptography.

ED25519

Deterministic signing. No nonce reuse risk. The same algorithm securing SSH and modern TLS.

MERKLE TREE

Every attestation record is part of an append-only tree. Tamper with one record and the entire chain breaks.

AES-256-GCM

Every private key encrypted at rest. The master key never touches the database.

OFFLINE VERIFICATION

Passports verify with only the CA public key. No network call to Vane required.

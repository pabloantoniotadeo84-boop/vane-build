Works where your agents already live

Three lines of code.
Any framework.

Drop Vane into your existing stack. No rearchitecting. No new infrastructure to operate. Every attested action is verifiable by anyone, forever.

LangChain 
import { VaneCallback } from "@vane.build/langchain";
 
const cb = new VaneCallback({
  agentId: process.env.VANE_AGENT_ID
});
 
await chain.call(input, { callbacks: [cb] });
OpenAI Agents SDK 
import { withPassport } from "@vane.build/openai-agents";
 
const agent = new Agent({ name: "researcher" });
 
export const run = withPassport(agent, {
  agentId: process.env.VANE_AGENT_ID
});
MCP 
import { vaneMiddleware } from "@vane.build/mcp-middleware";
 
// Drop in before your tool handlers
server.use(vaneMiddleware({
  agentId: process.env.VANE_AGENT_ID
}));
 
// Every tool call is now attested.
